<h1>A text based adventure game</h1>
<h2>Available Commands</h1>
<dl>
	<dt>Take</dt>
	<dd>Picks up item</dd>
	<dt>Inventory(inv) </dt>
	<dd>Shows current inventory </dd>
	<dt>Balance(bal)</dt>
	<dd>Shows current balance</dd>
	<dt>Clear</dt>
	<dd>Resets the screen</dd>
	<dt>Look</dt>
	<dd>Looks around</dd>
</dl>
